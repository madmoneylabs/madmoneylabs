## 第一章：Cloudflare WARP (速度与跨境办公首选)

### 1.1 基础配置（解决连不上的问题）
* **下载地址：** [https://1.1.1.1/](https://1.1.1.1/) (官方原版)
* **关键设置（避坑点）：**
    1.  打开 WARP 客户端，点击 **[小齿轮设置] -> [偏好设置] -> [高级]**。
    2.  **隧道设置：** 将协议手动更改为 `MASQUE` (如果已更新) 或 `UDP over TCP`。
    3.  **MTU 调整：** 在 [连接] 选项中，手动将 MTU 更改为 **1280**（这是解决网页加载慢、视频卡顿的秘诀）。
    4.  **IPv6 开关：** 务必开启“支持 IPv6”，这是 2026 年最快的“直连车道”。

### 1.2 进阶操作：优选 IP (Endpoint)
如果默认连接只有几百 KB/s，请按此操作：
1.  下载 **CloudflareSpeedTest** 工具（GitHub 搜索）。
2.  运行脚本，筛选出延迟低于 100ms 的 IP 地址。
3.  在 WARP 设置的 **[端点 (Endpoint)]** 栏中，输入优选后的 IP（例如：`162.159.x.x:2408`）。

## 第二章：Microsoft Edge 安全网络 (极简办公方案)

### 2.1 开启步骤
1.  打开 Edge 浏览器，点击右上角 **[...] -> [设置]**。
2.  左侧菜单点击 **[隐私、搜索和服务]**。
3.  向下滑动找到 **[Microsoft Edge 安全网络]**。
4.  **模式选择：** 勾选 **“优化”**。
    * *注：不要选“始终”，否则会浪费流量。选“优化”后，系统会自动判断：访问百度走直连，访问 ChatGPT 走 Azure 加速隧道。*

### 2.2 流量说明
* **额度：** 2026 年默认每月 15GB-30GB。
* **局限：** 仅限浏览器内流量。如果需要 Outlook 客户端或 Telegram 加速，请配合 WARP 使用。

## 第三章：Proton VPN (极端环境“救生圈”)

### 3.1 协议强制切换 (核心步骤)
Proton 默认设置很难直连，必须手动干预：
1.  打开 App -> **[Settings] -> [Protocol]**。
2.  取消勾选“Smart”，手动选择 **[Stealth]**。
3.  *Stealth 协议会将流量伪装成 HTTPS 网页请求，专门对付深度包检测。*

## 第四章：常见报错排除 (老专家贴士)

| 现象 | 解决方法 |
| :--- | :--- |
| **连接一直显示“正在连接”** | 检查系统时间。误差超 30 秒证书即失效。 |
| **能上 YouTube 但打不开 ChatGPT** | IP 被标记。尝试 WARP 优选 IP 或 Edge 优化模式。 |
| **UDP 阻断 (视频刚开始快随后断开)** | 强制切换到 **TCP 模式**，或将端口改为 **443**。 |

## 第五章：2026 避雷清单

1.  **拒绝“流氓加速器”：** 凡是名字带“旋风、极速、小飞机”且满大街广告的，基本都是实名采集器，避开。
2.  **不买个人流量包：** 尽量利用大厂基建。个人搭建的“小机场”安全性极低。
3.  **定期清理缓存：** 切换方案时，建议清除浏览器 DNS 缓存（`chrome://net-internals/#dns`）。
